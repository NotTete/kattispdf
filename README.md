# KattisPDF

KattisPDF is a Python library and tool that allows to convert problem statements from [Kattis](https://open.kattis.com) to pdf.

## Installation

To install KattisPDF just use `pip`:

```
pip install kattispdf
```

It is necessary to install a $\LaTeX$ processor such as [TeX Live](https://tug.org/texlive/) or [MiKTeX](https://miktex.org/). As [PyLaTeX](https://github.com/JelteF/PyLaTeX/tree/master)
